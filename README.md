# AccountX Cycle Count Prototype

This repository contains the HTML prototype for the AccountX Inventory Control Cycle Count specification.

## Current prototype

- HotelX Account Management home screen recreated in HTML and CSS
- Responsive sidebar, header, module cards and orange AccountX theme
- Clickable Inventory Control module prepared for the Cycle Count screens
- Hamburger menu interaction included

## GitHub Pages

The included GitHub Actions workflow deploys the site as a static GitHub Pages website.

Expected website address:

`https://edwardd767.github.io/cyclecount/`

If GitHub Pages is not yet enabled, open:

**Repository Settings → Pages → Build and deployment → Source → GitHub Actions**

## Accuracy note

The layout is closely matched to the supplied screenshot. The screenshot did not contain the original HotelX icon and background asset files, so the prototype uses equivalent Material Symbols and CSS-generated background artwork. Replacing them with the original assets later will allow a fully pixel-identical implementation.
